    • Motor Characterisation

    The following images are taken from my presentation regarding the design of a motor I created as apart of Monash Motorsport.

    The project entailed designing a motor with geometry that is optimised to produce maximium performance in competition based on a cost function that considers factors such as weight, torque, thermals, etc..

    To optimise for this cost function electromagnetic simulation was undergone using Ansys Maxwell to model and sweep different motor geometries to find the optimal design.

    The motor shaft was then modeled in SolidWorks and tested using the FEA software Ansys Maxwell, where load cases were simulated to validate whether the shaft could withstand operating conditions.

    Moreover, a machining plan of the shaft was created to ensure the design is feasibly machinable given the tools and sponsors of the team.

    Finally, CAD for the entire motor was created using the optimised geometry, alongside an assembly plan which details how the motor should be construed and ensures construction feasabillity.
